# Fitness-Tracker



### Description
>The purpose of this application is to track all exercises that the user inputs. The user has the ability to add on to their previous exercises or input new excercises they make.

### Technologies Used

>- Morgan
>- Mongo Database
>- Express
>- Mongoose
>- Node.js
>- robo 3T
